# Deutschgram Call

Лёгкий веб-мессенджер для личного общения: личные чаты, аудиозвонки и видеозвонки прямо в браузере.

## Что внутри

- PHP API без внешних зависимостей.
- SQLite для хранения пользователей, диалогов, сообщений и WebRTC-сигналов.
- Чистый HTML/CSS/JavaScript без сборки.
- WebRTC для аудио- и видеосвязи между двумя участниками.

## Как запустить в WAMP

1. Откройте проект по адресу `http://localhost/deutschgram/`.
2. В первом браузере войдите под одним именем, во втором под другим.
3. Разрешите доступ к микрофону и камере.
4. Выберите человека в списке слева, откройте диалог и нажмите `Аудио` или `Видео`.

## Ограничения MVP

- Сейчас нет паролей и полноценной авторизации.
- Диалоги только один на один.
- Для звонков используется только STUN. В сложных сетях позже стоит добавить TURN-сервер.
- Это веб-версия, без push-уведомлений и офлайн-режима.
